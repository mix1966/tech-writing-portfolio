What is GDSL Code

GDSL Code Elements

GDSL is a simple scripting language consisting of:

  • If/Else conditional statements
  • And/Or boolean statements
  • Assignment (=), comparison (>=, etc.) and mathematical (+, -, etc.) operators
  • Output literals (Output.Red, Output.Green and Output.Blue)
  • Data layer names. These are server-side image and data files (e.g. LandCover, ContinentalUSElevation)
  • Variables
